# Poe Creator Guide

Copy for LLMView as Markdown

Welcome to the Poe Creator Platform documentation. Here you'll find everything you need to build, deploy, and monetize AI applications on Poe.

## [What is Poe?](https://creator.poe.com/docs\#what-is-poe)

Poe is a platform that allows you to create and share AI-powered applications with millions of users. Whether you're building chatbots, tools, or interactive experiences, Poe provides the infrastructure and distribution to make your creations successful.

## [Getting Started](https://creator.poe.com/docs\#getting-started)

Choose your path:

- **[Prompt Bots](https://creator.poe.com/docs/prompt-bots/how-to-create-a-prompt-bot)** \- Configure a base bot with custom instructions
- **[Server Bots](https://creator.poe.com/docs/server-bots/quick-start)** \- Build bots on a custom server with full programmatic control
- **[Script Bots](https://creator.poe.com/docs/script-bots/quick-start)** \- Create Python-based bots hosted on Poe through vibe-coding
- **[Canvas Apps](https://creator.poe.com/docs/canvas-apps/canvas-app-quick-start)** \- Create interactive web applications
- **[External Applications](https://creator.poe.com/docs/external-applications/external-application-guide)** \- Integrate Poe with your existing applications
